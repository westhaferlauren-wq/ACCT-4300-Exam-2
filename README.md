<section class="topic" id="ch8">

<h2>Chapter 8 – Fraud and Errors</h2>

<div class="card">
<h3>Process / Overview</h3>
<ul>
<li>Fraud is an intentional act to obtain an unjust advantage.</li>
<li>Errors are unintentional mistakes in accounting records.</li>
<li>Both can cause misstatements in financial statements.</li>
<li>Internal controls help prevent and detect fraud.</li>
</ul>
</div>

<div class="card">
<h3>Key Concepts</h3>
<ul>
<li>Fraud Triangle: Pressure, Opportunity, Rationalization</li>
<li>Fraud Diamond: Adds capability to the fraud triangle</li>
<li>Occupational fraud categories: asset misappropriation, corruption, financial statement fraud</li>
</ul>
</div>

<div class="card">
<h3>Key Threats</h3>
<ul>
<li>Theft of cash receipts</li>
<li>Fictitious vendors</li>
<li>Payroll fraud</li>
<li>Expense reimbursement fraud</li>
<li>Manipulation of financial statements</li>
</ul>
</div>

<div class="card">
<h3>Controls</h3>
<ul>
<li>Segregation of duties</li>
<li>Authorization procedures</li>
<li>Physical access restrictions</li>
<li>Internal audits</li>
<li>Account reconciliations</li>
</ul>
</div>

<div class="card">
<h3>Financial Statement Impact</h3>
<ul>
<li>Cash theft → Cash understated</li>
<li>Fake revenue → Revenue overstated</li>
<li>Expense fraud → Expenses overstated</li>
</ul>
</div>

<div class="card">
<h3>Micro-Check</h3>
<p>What are the three elements of the fraud triangle?</p>

<button onclick="toggleAnswer('fraudtriangle')">Reveal guidance</button>

<div id="fraudtriangle" class="answer">
Pressure, Opportunity, Rationalization
</div>

</div>

</section>

<h2>Chapter 9 – Computer Fraud and Abuse Techniques</h2>

<div class="card">
<h3>Process / Overview</h3>
<ul>
<li>Organizations rely heavily on information systems.</li>
<li>Cybercriminals attempt to exploit vulnerabilities.</li>
<li>Security controls protect data confidentiality, integrity, and availability.</li>
</ul>
</div>

<div class="card">
<h3>Types of Attacks</h3>
<ul>
<li>Social engineering</li>
<li>Malware</li>
<li>Password attacks</li>
<li>Data theft</li>
<li>Denial-of-service attacks</li>
</ul>
</div>

<div class="card">
<h3>Examples of Malware</h3>
<ul>
<li>Viruses</li>
<li>Worms</li>
<li>Trojan horses</li>
<li>Ransomware</li>
<li>Spyware</li>
</ul>
</div>

<div class="card">
<h3>Key Threats</h3>
<ul>
<li>Unauthorized access to systems</li>
<li>Data breaches</li>
<li>System downtime</li>
<li>Loss of sensitive financial data</li>
</ul>
</div>

<div class="card">
<h3>Controls</h3>
<ul>
<li>Strong password policies</li>
<li>Multi-factor authentication</li>
<li>Firewalls</li>
<li>Intrusion detection systems</li>
<li>Encryption and secure backups</li>
</ul>
</div>
