<h2>Chapter 8 – Fraud and Errors</h2>

<div class="card">
<h3>Key Takeaways</h3>
<ul>
<li>Fraud is intentional deception to obtain an unjust advantage.</li>
<li>The fraud triangle includes pressure, opportunity, and rationalization.</li>
<li>Internal controls reduce opportunities for fraud.</li>
</ul>
</div>

<div class="card">
<h3>Examples</h3>
<ul>
<li>Employee stealing cash receipts</li>
<li>Manipulating accounting records</li>
<li>Creating fictitious vendors</li>
</ul>
</div>

<div class="card">
<h3>Common Confusions</h3>
<ul>

</ul>
